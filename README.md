# ذاكرة الجبل — Memory of the Mountain

> تجربة رقمية سينمائية غامرة مكرَّسة لحفظ وعرض تراث منطقة الونشريس وتيسمسيلت — كتب، حكايات، أعلام، خرائط، أصوات، فيديوهات، وقاموس تراثي.

تطبيق ويب يحاكي تجربة تطبيق جوّال أصلي (Native Mobile App) بدعم كامل للعربية واتجاه RTL، مبني كـ PWA قابل للتثبيت على الشاشة الرئيسية.

---

## 🛠️ التقنيات

| المجال | التقنية |
|---|---|
| Framework | TanStack Start (React 19 + Vite 7) |
| Routing | TanStack Router (file-based) |
| Styling | Tailwind CSS v4 + OKLCH design tokens |
| Animations | Framer Motion |
| Icons | Lucide React |
| PWA | Web App Manifest + Installable |
| TTS | Web Speech API (Arabic) |
| Language | Arabic / French / English / Amazigh (UI ready) |

## 🎨 نظام التصميم

- **خلفية ليلية دافئة** (`oklch(0.16 0.018 50)`) مع توهّجات إشعاعية ذهبية وطينيّة.
- **لون رئيسي**: ذهب تراثي `--gold` + طين `--clay` + أخضر عميق `--emerald-deep`.
- **Glassmorphism** ذو طبقتين (`.glass` و`.glass-strong`).
- **Grain overlay** سينمائي و**shadow-luxe** متعدّد الطبقات.
- **Typography**: Tajawal (عربي)، Cormorant Garamond (Display)، Amiri (نصوص أدبية)، Inter (لاتيني).
- **Safe area** كاملة (top + bottom) لمحاكاة شاشة iPhone.

## 📚 المحتوى المضمَّن

### ٥ كتب PDF أكاديمية حقيقية
- أسطورة سيدي رابح بوقبرين
- الشعر الشعبي الملحون في الونشريس
- المظاهر الفكرية في الونشريس (ق. ٥م)
- قبائل الونشريس في العهد الروماني (جزآن)

### ٤ حكايات كاملة بنصوص أصلية
- أسطورة سيدي رابح بوقبرين
- نداء العود في المساء (الشعر الملحون)
- ظلال القرن الخامس
- خيوط الحرف (النسيج التقليدي)

### ٣ سير ذاتية لأعلام المنطقة
- الشيخ المنوّر الونشريسي (فقيه ومؤرّخ)
- سي عبد القادر الملحون (شاعر شعبي)
- للاّ خديجة بنت الجبل (حافظة ذاكرة شعبية)

### ١٧ مدخلًا في القاموس التراثي
البرنوس، الحايك، الكسكسي، الشخشوخة، المطمورة، الزاوية، القصبة، الدوّار، الكانون، الرحى، المنسج، العقدة، التويزة، الوعدة، الزربيّة، الخزف، الحلفاء — مع نطق وأمثلة وتصنيف.

### ٨ أحداث في الخط الزمني
من مقاومات قبائل الونشريس ضد الرومان (٤٠م) إلى استحداث ولاية تيسمسيلت (١٩٨٤).

### ٦ مواقع على الخريطة التفاعلية
قمّة سيدي عمر، قصبة تيسمسيلت، مغارة الرمّان، مقام سيدي رابح، قرية بني شقران، آثار Tigava.

### ٥ مقاطع صوتية + ٢٠ فيديو يوتيوب
شعر ملحون، أناشيد، موسيقى، حكايات مسموعة، وثائقيات.

## ✨ المزايا الرئيسية

### تجربة التطبيق
- 🎬 **شاشة إقلاع سينمائية** + Onboarding متحرّك
- 📱 **PhoneFrame** على الديسكتوب لمحاكاة الجوّال
- 🔘 **Bottom Navigation** بـ 5 تبويبات + Spring animation للمؤشر النشط
- 🌐 **دعم RTL كامل** مع طباعة عربية أنيقة
- 📥 **Installable PWA**: manifest.json + أيقونات + InstallPrompt تلقائي

### القراءة
- 📖 **قارئ PDF داخلي** عبر iframe مع زر تنزيل
- 🔊 **Text-to-Speech عربي** في صفحة الحكاية (Web Speech API)
- 📤 **مشاركة الحكاية** عبر Web Share API + نسخ احتياطي للـ clipboard
- 🔖 أزرار حفظ ومشاركة عائمة

### الاستكشاف
- 🗺️ **خريطة تفاعلية** مع علامات نابضة + bottom sheet للتفاصيل
- ⏳ **خط زمني عمودي** متحرّك بسكرول fade-in
- 📕 **قاموس قابل للبحث** مع تصفية حسب الفئة ونطق صوتي
- 👥 **سير الأعلام** مع صور بورتريه مولّدة بالذكاء الاصطناعي

### الصوت والوسائط
- 🎵 **مشغّل صوت** بنمط Spotify مع Mini Player عائم وشريط تقدّم
- 📺 **معرض فيديو** + مشغّل YouTube مدمج
- 🖼️ **معرض صور** عبر الكتب والمجموعات

### الإعدادات
- 🌓 الوضع الداكن / الفاتح
- 🔤 تكبير/تصغير الخطّ (14–22 px)
- 🌍 تبديل اللغة (ar/fr/en/ⵜⵎⵣⵗⵜ)
- 🔔 تفعيل الإشعارات والتنزيل التلقائي

### الإشعارات
- 🔔 مركز إشعارات بـ 4 أنواع: حكاية، كتاب، وسائط، تذكير
- 🟡 مؤشّرات قراءة/عدم قراءة بصرية

## 📦 الحالة الحالية

التطبيق **frontend-only** وجاهز لربط backend مستقبلًا (Lovable Cloud / Supabase) للميزات التالية:
- مصادقة حقيقية (تسجيل/دخول)
- مزامنة المفضّلة والإشارات المرجعية عبر الأجهزة
- التعليقات والتقييمات
- إشعارات Push حقيقية
- لوحة تحكّم إدارية (Admin Dashboard)

البنية الحالية معدّة لاستقبال هذه الإضافات دون إعادة هيكلة.
